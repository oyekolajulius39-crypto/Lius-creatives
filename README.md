### Changing Colors
Edit the CSS variables in `css/style.css`:
```css

  --primary-blue: #0A2463;
  --accent-orange: #FF6B35;
  /* Add your custom colors */
```

In `portfolio.html`, duplicate a portfolio item and update:
- `data-category` attribute
- Title and description
- Background gradient

### Modifying Services
Edit `services.html` to add/remove services in the service detail sections.
